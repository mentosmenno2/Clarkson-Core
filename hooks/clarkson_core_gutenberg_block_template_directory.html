Filter: clarkson_core_gutenberg_block_template_directory

apply_filters( 'clarkson_core_gutenberg_block_template_directory', '/templates/partials/blocks/gb/' ) → {string}

Allows theme to overwrite the directory in which Clarkson Core automatically loads twig templates for blocks.
Parameters:
Name Type Description
'/templates/partials/blocks/gb/' string Directory that will be used to automatically load gutenberg block templates.
Since:
  • 0.4.0
Source:
  • lib/clarkson-core-gutenberg-block-type.php, line 33
Returns:
Directory to use for autoloading twig templates.
Type
string
Example
// Overwrite the Gutenberg block template directory.
add_filter(
 'clarkson_core_gutenberg_block_template_directory',
 function() {
     return '/templates/gutenberg/blocks/';
 }
);
