Clarkson_Core_Gutenberg_Block_Manager

Intercepts 'the content' filter to allow overriding of the rendering functions of Gutenberg blocks. This allows us to use twig for block rendering.
Clarkson_Core_Gutenberg_Block_Type

This custom block overwrites the render callback to use twig files. It is automatically injected into all registered blocks at render time.
Clarkson_Core_Objects

This class is used to convert WordPress posts, terms and users into Clarkson Objects.
